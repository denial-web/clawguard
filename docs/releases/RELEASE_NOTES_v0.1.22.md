# ClawGuard v0.1.22

This release makes the published npm CLI easier to invoke across npm versions.

## Added

- `clawguard --version` and `clawguard -v`.
- Test coverage for CLI version output.

## Updated

- README, docs, CLI help, and web demo commands now use the robust scoped-package form:
  `npx --package @denial-web/clawguard clawguard ...`
